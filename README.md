## How to start the cloud version of the pictionary app

Make sure to have nodejs installed

Enter in the Cloud folder and start the application with the command:

```
node server.js
```

Then you can use the URL https://[localhost or ip of the server]:8081


## How to start the edge version of the pictionary app

Enter in the Edge\socket.io-p2p-master\examples\Pictionary folder and start the application with the command:
```
node server.js
```

Then you can use the URL https://[localhost or ip of the server]:8080
